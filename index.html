Academic Projects

DEVELOPMENT OF A PLASTIC SORTING AND SHREDDING MACHINE | INDIVIDUAL
(Nov. 2024 - Jul. 2025)
Designed and developed a mechatronics-based plastic sorting system to automatically classify and sort plastic waste. The system identifies PET bottles, beverage cans, shampoo bottles (HDPE), and yogurt cups (PP) using image processing and a ResNet18 deep learning model running on a Raspberry Pi. Classified items are directed into appropriate bins via an Arduino-controlled motorized mechanism.

Key Achievements:
  • Achieved 98% sorting accuracy in real-time testing.
  • Developed a custom dataset and implemented Python + OpenCV for real-time detection.
  • Designed and fabricated the sorting mechanism & bin selector in SolidWorks.
  • Integrated stepper motor control for automated sorting operation.
  • User-friendly: Insert an item, press a button, and the system classifies & sorts automatically.
Applications:

Schools, supermarkets, offices, households, and urban areas for efficient waste segregation.

Future Enhancements:
  • Add shredding mechanism for volume reduction.
  • Integrate reward system for user engagement.
Technologies & Tools:

Python, PyTorch, OpenCV, Raspberry Pi, Arduino, SolidWorks, Machine Learning, Image Processing.

SHELL & TUBE HEAT EXCHANGER | GROUP
(2024 - Present)
A comprehensive project focusing on the design, modeling, and analysis of a shell and tube heat exchanger. This project aims to compare the effectiveness of parallel flow and counter flow configurations and test the effectiveness of two different baffle types in enhancing heat transfer efficiency.

  • Skills: SolidWorks, Computational Fluid Dynamics (CFD), Mesh Convergence Analysis
  • Project Objectives: Design and model the shell and tube heat exchanger using SolidWorks, Extract fluid domains inside the shell and tube, Perform mesh convergence and evaluate mesh quality, Conduct four separate CFD simulations to compare baffle types with different flow configurations, Obtain and analyze outlet temperatures, flow patterns, temperature, and pressure profiles inside the heat exchanger.
  • Methodology: Modeled the heat exchanger using SolidWorks, selecting suitable thicknesses and materials for the shell and tubes, Extracted fluid domains inside the shell and tube, Performed mesh convergence and evaluated mesh quality, Conducted four CFD simulations to compare baffle types with different flow configurations, Analyzed outlet temperatures, flow patterns, temperature, and pressure profiles.
  • Key Results: Comparison of parallel flow and counter flow configurations, Effectiveness of two different baffle types in enhancing heat transfer, Detailed report including screenshots, graphs, and tables illustrating the findings.
